# Prohibitions (Strictly Forbidden)

1. **Unauthorized Architecture Changes**: Do not add new classes or modify inheritance structures that are not defined in the design specifications. Any structural changes require explicit approval via the chat tab.

2. **Modifying `*.generated.h` Files**: Never manually edit *.generated.h files. These are automatically managed and generated by the Unreal Build Tool (UBT).

3. **Arbitrary Include Order Changes**: Do not deviate from the established include sequence. The `*.generated.h` include must always be the final entry.

4. **Modifying UPROPERTY Specifiers**: Use the exact UPROPERTY specifiers defined in the design documents. Do not change access levels (e.g. `BlueprintReadOnly` to `BlueprintReadWrite`) without permission.

5. **Writing Logic in Blueprints**: This is a C++ centric project. Implementation of core logic in Blueprints is strictly prohibited.

6. **Tick-based UI Updates**: Do not use NativeTick or Blueprint Tick for UI updates. All UI logic must be event-driven.

7. **Magic Numbers**: Hard-coded values are forbidden. Use static constexpr for internal constants or expose them via UPROPERTY for designer-facing values.
